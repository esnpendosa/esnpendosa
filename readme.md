## 🚀 Highlighted Projects

Berikut adalah beberapa proyek unggulan yang menunjukkan kompetensi saya dalam pengembangan sistem web, otomasi kecerdasan buatan (AI), analitik data, dan pembuatan plugin kustom:

### 🤖 AI & Automation Specialist
- **[JARVIS-CUSTOM](https://github.com/esnpendosa/JARVIS-CUSTOM)**
  Asisten AI berbasis suara (Voice Assistant) interaktif yang dilengkapi dengan antarmuka grafis (PyQt5 GUI) untuk menjalankan otomasi tugas lokal pada Windows, pengenalan suara (Speech Recognition), dan integrasi API pihak ketiga secara real-time.
  *Tech Stack:* `Python`, `PyQt5`, `SpeechRecognition`, `API Integration`
- **[aimousecontrol](https://github.com/esnpendosa/aimousecontrol)**
  Aplikasi kontrol navigasi kursor mouse nirkabel menggunakan deteksi gerakan tangan (hand gestures) berbasis Computer Vision untuk mendukung interaksi komputer tanpa sentuh.
  *Tech Stack:* `Python`, `OpenCV`, `MediaPipe`, `Computer Vision`

### 💻 Web Systems & Applications
- **[LONEWOLF-By-LINGKUP](https://github.com/esnpendosa/LONEWOLF-By-LINGKUP)**
  Platform internal manajemen perusahaan (enterprise ERP) yang dirancang untuk mengotomatisasi alur kerja, pengelolaan sumber daya, dan operasional bisnis internal di PT Lingkup Indonesia Corp.
  *Tech Stack:* `Laravel`, `PHP`, `MySQL`, `Bootstrap`
- **[RKASIR](https://github.com/esnpendosa/RKASIR)**
  Aplikasi kasir berbasis web (Point of Sale) real-time yang mempermudah pencatatan transaksi ritel, manajemen stok inventori, dan generate struk belanja digital secara instan.
  *Tech Stack:* `JavaScript`, `Node.js`, `HTML5`, `CSS3`
- **[ppdbsmkuma](https://github.com/esnpendosa/ppdbsmkuma)** & **[smkuma](https://github.com/esnpendosa/smkuma)**
  Portal Penerimaan Peserta Didik Baru (PPDB) serta Sistem Informasi Manajemen Sekolah terintegrasi untuk mendigitalisasi administrasi pendaftaran siswa secara efisien.
  *Tech Stack:* `PHP`, `Laravel`, `MySQL`, `Bootstrap`
- **[sistem-keluhan](https://github.com/esnpendosa/sistem-keluhan)**
  Platform pengaduan dan layanan keluhan masyarakat/karyawan terintegrasi yang memudahkan pengiriman laporan pengaduan secara real-time lengkap dengan dashboard status penanganan kasus.
  *Tech Stack:* `PHP`, `Laravel`, `Blade`, `MySQL`

### 🔌 Custom WordPress Plugins
- **[plugin-sertifikat-digital-wordpress](https://github.com/esnpendosa/plugin-sertifikat-digital-wordpress)**
  Plugin WordPress kustom untuk menjana (generate) sertifikat PDF digital secara otomatis berdasarkan database kelulusan peserta dan mengirimkannya langsung ke email pengguna.
  *Tech Stack:* `PHP`, `WordPress Core API`, `FPDF Library`
- **[plugin-laporan-monev-kurikulum-wordpress](https://github.com/esnpendosa/plugin-laporan-monev-kurikulum-wordpress)**
  Sistem monitoring dan evaluasi (Monev) kurikulum pendidikan terintegrasi WordPress untuk mempermudah pelaporan, rekapitulasi data, dan analisis performa akademik secara berkala.
  *Tech Stack:* `PHP`, `WordPress Core API`, `Custom MySQL Tables`

### 📊 Data Science & Machine Learning
- **[Kelurahan-Randuagun-Naive-Bayes-Welfare-Classification](https://github.com/esnpendosa/Kelurahan-Randuagun-Naive-Bayes-Welfare-Classification)**
  Sistem klasifikasi tingkat kesejahteraan sosial keluarga kurang mampu menggunakan algoritma Naive Bayes Classifier untuk membantu akurasi penyaluran bantuan sosial agar tepat sasaran.
  *Tech Stack:* `Go (Golang)`, `Naive Bayes`, `MySQL`
- **[Analisis-Sentimen-dan-Regresi-Ulasan-Metode-MULTINOMIAL-NAIVE-BAYES](https://github.com/esnpendosa/Analisis-Sentimen-dan-Regresi-Ulasan-Metode-MULTINOMIAL-NAIVE-BAYES)**
  Model analisis sentimen dan regresi terhadap data ulasan pelanggan menggunakan Multinomial Naive Bayes guna mengidentifikasi kecenderungan feedback positif atau negatif.
  *Tech Stack:* `Python`, `Jupyter Notebook`, `Scikit-Learn`, `Pandas`
- **[Presiksi-Obat-Jantung-Metode-Monte-Carlo](https://github.com/esnpendosa/Presiksi-Obat-Jantung-Metode-Monte-Carlo)**
  Aplikasi proyeksi dan prediksi persediaan obat jantung di apotek menggunakan metode simulasi Monte Carlo untuk mencegah kekosongan stok obat krusial.
  *Tech Stack:* `PHP`, `Monte Carlo Simulation`, `MySQL`
